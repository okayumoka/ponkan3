# Ponkan3


## build (main programs)

```bash
# install packages
$ npm install

# case of build only
$ npm run build
# case of develop
$ npm start
```

## build (manual pages)

Using [MkDocs](https://github.com/mkdocs/mkdocs/).

```bash
# install mkdocs
pip install mkdocs

# case of only build
$ manual/scripts/build.sh
# case of develop
$ manual/scripts/start_dev_server.sh
```
